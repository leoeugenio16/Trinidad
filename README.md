# Tarjeta XV - Trinidad

Invitación digital estática lista para GitHub Pages.

## Archivos principales

- `index.html`: estructura de la tarjeta.
- `styles.css`: diseño visual.
- `script.js`: contador, música, animaciones y copiar link.
- `evento-trinidad.ics`: archivo para agregar el evento al calendario.
- `assets/`: fotos y música.

## Qué cambiar

1. Reemplazar las imágenes:
   - `assets/hero.jpg`
   - `assets/foto-1.jpg`
   - `assets/foto-2.jpg`
   - `assets/foto-3.jpg`
   - `assets/foto-4.jpg`
   - `assets/foto-5.jpg`
   - `assets/foto-6.jpg`

2. Reemplazar links:
   - WhatsApp: buscar `5492600000000` en `index.html`.
   - Google Forms: buscar `https://forms.gle/REEMPLAZAR_FORMULARIO`.
   - Google Maps: buscar `https://maps.google.com`.
   - YouTube: cambiar el `src` del iframe.

3. Cambiar lugar y hora:
   - Buscar `Salón a confirmar` y `21:00 hs`.

## Publicar en GitHub Pages

1. Crear un repo nuevo, por ejemplo `xv-trinidad`.
2. Subir estos archivos.
3. Ir a Settings > Pages.
4. En Source elegir `Deploy from a branch`.
5. Branch: `main`, folder: `/root`.
6. Guardar.

El link quedará parecido a:

`https://tuusuario.github.io/xv-trinidad/`
